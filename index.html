Our React workshop is a great choice for professional developers who would like to jump - head first, into the exciting world of frontend development with React.


Is it for me?

If you're an frontend developer , you will have all the knowledge and tools to upgrade to modern React.

If you're a Java/.NET developer , we believe that you are making a great decision, prepare yourself for a great ride.

Students must have working knowledge of HTML and JavaScript, and another DOM manipulation library such as jQuery, or experience working on server-side web development (Java, PHP, etc).

What Will I Learn?

We will review the tools every frontend developer should know, and dive into implementing a scalable React application.

Weather you've worked with other frameworks, or not, the strength of React will amaze you.


You will experience that React embraces:


Using ES6 for better Javascript

Using Modern Modular CSS

Single page, progressive web applications

The magic and powers of reactivity

State management, Flux, Redux and Mobx

Best Practices for modern frontend development
